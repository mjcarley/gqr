A type defining the quadrature rules available. A quadrature rule is defined as a basic type possibly combined with a singularity. For example GQR\_\-GAUSS\_\-LEGENDRE $|$ GQR\_\-GAUSS\_\-HYPERSINGULAR specifies a Gauss-Legendre rule which handles singularities up to and including second order.
